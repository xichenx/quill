# Changelog

## [Unreleased]

### Added

- Continuous scroll mode — pages flow seamlessly without gaps
- Remember reading position — page, zoom, rotation, and scroll mode are restored on reopen
- Scroll mode toggle button in toolbar and View menu
- Text selection and copy — select text on PDF pages and copy with Ctrl+C
- Editable bookmarks — double-click bookmark labels to rename them
- Export pages as images — save current page or all pages as PNG/JPEG

## [0.1.0] - unreleased

### Added

- Initial release of Quill PDF Reader
- PDF rendering with zoom, fit-to-width, and fit-to-page modes
- Dark mode / light mode toggle
- Multi-tab reading
- Full-text search with match highlighting
- Thumbnail sidebar for page navigation
- Keyboard shortcuts for all major actions
- Drag & drop PDF files to open
- Recent files list
- Print support
- Page rotation (90° CW)
- Custom frameless window with native title bar controls
